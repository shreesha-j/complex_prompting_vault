## Field Type Guidelines

Choose the appropriate field type for each field:

- **CoreField**: Generic field with custom display_type
  - Use for: text inputs, textareas, custom fields
  - Set display_type: "textbox", "textarea", "dropdown", "radio", "checkbox"
  
- **FIELD.Email**: Email field with built-in validation
  - Use for: email addresses
  - Automatically validates email format
  
- **FIELD.Phone**: Phone number field
  - Use for: phone numbers
  - Validates phone number format
  
- **FIELD.Date**: Date picker field
  - Use for: dates (DOB, expiry dates, etc.)
  - Provides date picker UI
  
- **FIELD.FILE**: File upload field
  - Use for: document uploads
  - Handles file upload functionality
  
- **FIELD.PAN**: PAN number field
  - Use for: PAN card numbers
  - Validates PAN format
  
- **FIELD.AADHAAR**: Aadhaar number field
  - Use for: Aadhaar card numbers
  - Validates Aadhaar format
