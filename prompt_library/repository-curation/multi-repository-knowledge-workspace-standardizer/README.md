# Multi-Repository Knowledge Workspace Standardizer

This folder isolates one prompt concept so you can understand what it is for without mixing it with unrelated prompt assets.

## Why this folder exists

Captures the full workflow used to reorganize the `Notes`, `Context-Database_Handler`, and `complex_prompting_vault` repositories so the same standards can be reused later.

## What it contains

A master prompt plus reusable instruction templates for curating note, context, and prompt repositories with README/index coverage, manifests, migration maps, commit codes, and workspace-level indexing.

## Files in this folder

- `prompt.md`: Vault-authored master prompt for multi-repository knowledge workspace curation.
- `template-instruction.md`: Full reusable instruction template with the standard prompt inlined and precedent repo paths prefilled.
- `quick-template-instruction.md`: Shorter version for fast agent chats.

## Notes

These files were created directly inside the vault during the Jess/Git repository standardization work, so they do not have an external raw-import source.
